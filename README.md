# Jarkom-Chatroom

1. Run ChatServer.java pada src/main/java/com/chatroom/backend untuk memulai Server dari Chatroom App.
2. Run Application.java pada src/main/java/com/chatroom untuk memulai sebuah memulai sebuah instance dari Client.
3. Masukkan IP dari komputer yang menjadi Server jika melakukan koneksi dari komputer yang berbeda.
4. Jika tidak, isi dengan 'localhost' saja bisa.
